# Vote-System
Vote system Api

## Environment
require for NodeJS and NPM environment, based on mongoDB

## Install
```bash
$ cd 项目路径n
$ npm stall
```

# Develop
```bash
$ npm run start
```

# Test
```bash
$ npm run test
```

# Deploy
```bash
$ npm run prod
```

# Post api-doc
```bash
$ npm run apidoc
```
