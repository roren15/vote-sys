# Vote-System
Vote system Api

## Environment
develop: require for NodeJS and NPM environment, based on mongoDB
docker: require for docker and docker-compose

# Docker Deploy
```bash
$ cd project-path
$ docker-compose up -d
```

## Install
```bash
$ cd project-path
$ npm install
```

# Mocha Test
```bash
$ npm run test
```

# Docker Image Build
```bash
$ cd project-path
$ sh dockerbuild.sh
```

# Develop
```bash
$ npm run start
```

# Post api-doc
```bash
$ npm run apidoc
```
