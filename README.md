# Vote-System
Vote system node Api, serviced for basic vote and its manage

## Environment
Develop: require for NodeJS and NPM environment, based on mongoDB ;
Docker: require for docker and docker-compose

# Docker Deploy
```bash
$ cd project-path
$ docker-compose up -d
```

## Install
```bash
$ cd project-path
$ npm install
```

# Mocha Test
```bash
$ npm run test
```

# Docker Deploy
```bash
$ cd project-path
$ sh dockerbuild.sh
```

# Develop
```bash
$ npm run start
```

# Post api-doc
```bash
$ npm run apidoc
```
